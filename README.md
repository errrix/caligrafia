# Каллиграфия

Генератор печатных прописей формата A4. Приложение работает в браузере,
позволяет ввести фразу, настроить размер текста и распечатать лист или сохранить
его в PDF через системный диалог браузера.

## Быстрый старт

```bash
npm install
npm run dev
```

Production-сборка:

```bash
npm run build
```

Проверки:

```bash
npm run lint
```
